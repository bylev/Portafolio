# Portafolio Web - Michelle Camara

Portafolio personal desarrollado con HTML, CSS y JavaScript puro.

Este sitio presenta mi perfil profesional, habilidades tecnicas, certificaciones, proyectos destacados y galeria de participaciones en eventos.

## Link del Proyecto
- GitHub: `https://bylev.github.io/Portafolio/`

## Vista General

- Diseño responsivo (desktop, tablet y mobile)
- Modo oscuro / modo dia con persistencia en `localStorage`
- Soporte bilingue (ES / EN)
- Carrusel ciclico para certificaciones
- Carrusel ciclico para galeria de proyectos
- Seccion de contacto con LinkedIn, GitHub y correo

## Como Ejecutarlo Localmente

### Opcion 1: abrir directo

Abrir `index.html` en el navegador.

### Opcion 2: servidor local recomendado

Si tienes Python instalado:

```bash
python -m http.server 5500
```

Luego abrir:

```http://localhost:5500/Portafolio/```
## Personalizacion Rapida

- Nombre y rol principal: `index.html`
- Colores/tema visual: `style.css` (variables en `:root`)
- Textos ES/EN: `script.js` objeto `translations`
- Links de contacto: `index.html` seccion `#contact`
- Imagenes de galeria: carpeta `img/`

## Highlights Implementados

- Cambio de idioma con textos dinamicos via `data-i18n`
- Toggle de tema con estado persistente
- Efecto visual de aura siguiendo el mouse
- Carruseles con navegacion por flechas y bucle infinito
